# Next.js Practice Project

A modern practice project built with **Next.js**, **Tailwind CSS**, and **daisyUI**.  
This project was created to improve my understanding of routing, dynamic pages, reusable components, responsive layouts, and UI styling in Next.js.

This is a simple multi-page web application developed as part of my Next.js practice. It includes a clean homepage, blog listing page, blog details page with dynamic routing, developers page, contact page, and theme toggle support.

The project focuses on learning how to structure a Next.js application properly while building responsive and visually appealing interfaces.

## Features

- Built with **Next.js App Router**
- Responsive UI with **Tailwind CSS**
- Pre-styled components using **daisyUI**
- Dynamic blog details page using route parameters
- Reusable page sections and cards
- Theme toggle functionality
- Simple and clean layout for practice and learning

- **Blogs Page**

- **Contact Page**

- **Tailwind CSS**
- **daisyUI**

## Project Goals

This practice project was created to learn and improve the following skills:

- File-based routing in Next.js
- Dynamic routing with route parameters
- Creating reusable UI components
- Responsive design using Tailwind CSS
- Using daisyUI with Tailwind
- Basic theme switching
- Organizing project files in a clean structure
